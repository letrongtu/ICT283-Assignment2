Detailed Description

Encapsulates weather data including date, time, wind speed, solar radiation, and ambient air temperature.

Author
Trong Tu Le
Version
1 Each WeatherRecord object holds a date and time of recording, along with wind speed, solar radiation, and ambient air temperature. The class provides both getters and setters for all data fields to ensure encapsulated and safe access to the data.
