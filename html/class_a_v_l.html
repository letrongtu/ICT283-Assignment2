Detailed Description

template<class T>
class AVL< T >

Template class for an AVL Tree, which is a self-balancing Binary Search Tree.

Author
Trong Tu Le
Version
1 turn BST to AVL tree added function: height, getBalance, rightRotate, leftRotate, max modified function: insertHelper
Template Parameters
TData type of the elements in the AVL Tree.

The AVL tree maintains its balance using rotations to ensure the heights of the subtrees of any node differ by at most one.
